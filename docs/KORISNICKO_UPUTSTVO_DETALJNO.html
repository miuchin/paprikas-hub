1) Brzi start

  1. Pokreni server: python3 server/paprikas_server.py --host 127.0.0.1 --port 8015
  2. Otvori http://127.0.0.1:8015/
  3. Za spasavanje jela koristi Početna → Rescue / Atlas.
  4. Za dopunu baze recepata koristi Početna → 🧰 Content Studio.
  5. Za reference koristi Biblioteka i Trikovi & kuća.

2) Početna i glavni ulazi

🏠 Početna

Početna je komandni centar. Cilj je da korisnik ne skroluje kroz dugačke tabove nego da iz Početne odmah uđe u potrebnu radnu stanicu.

📖 Recipe Atlas

Baza recepata, ručni unos, import/export, pregled i štampa recepata.

🧰 Content Studio

Serijski unos novih recepata: prompt → batch → extract titles → auto-fix → merge → export.

📚 Biblioteka / Trikovi

Reference, enciklopedija, kuhinjski i kućni trikovi, fleke i mirisi.

3) Recipe Atlas i rad sa receptima

  • Recipe Atlas je ciljna baza recepata.
  • Manual add koristi se za jedan recept ili ručne ispravke.
  • Import prima finalni NDJSON koji dolazi iz Content Studio merge izlaza.
  • Inventory export služi za dedup pri narednim promptovima.

4) Content Studio (10×100)

Redosled rada

  1. Izaberi batch.
  2. Build prompt (auto).
  3. Open helper → Copy.
  4. Nalepi prompt u ChatGPT.
  5. Dobijeni NDJSON nalepi nazad u Studio ili uploaduj.
  6. Extract titles.
  7. Posle svih batch-eva: Merge titles → Auto-fix + Merge.

Rezultati

  • recipes_1000_merged.ndjson
  • recipes_1000_merged_report.json
  • invalid_lines i duplicates fajlovi
  • reports_bundle.zip
Soft accept prazna kolicina je koristan kad AI ostavi sastojak bez količine. Recept prolazi, ali zadržava se pregled kroz report.

6) Troubleshooting

  • Server ne kreće: proveri terminal traceback; koristi isti Python i isti folder projekta.
  • UI izgleda staro: uradi Ctrl+Shift+R.
  • Batch ne prolazi: proveri NDJSON format i uključi Auto-fix.
  • Previše duplikata: koristi inventory export + all-previous titles.
