# Paprikaš Hub — Help / FAQ (v5.17.5)

## Brzi start
- Pokreni server: `python3 server/paprikas_server.py --host 127.0.0.1 --port 8015`
- Otvori: `http://127.0.0.1:8015/`
- Početna = komandni centar

## Pravilo rada
- problem u jelu → Rescue / Atlas
- dopuna baze recepata → Content Studio
- reference i trikovi → Biblioteka / Trikovi
- dokumentacija → Korisničko uputstvo + Help / FAQ

## Najčešći problemi
- stara verzija UI → `Ctrl+Shift+R`
- merge ne prolazi → proveri NDJSON + Auto-fix
- duplikati → inventory titles + all-previous titles
- ne radi docs link → proveri da otvaraš novu verziju ZIP-a
