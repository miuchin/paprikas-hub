🚀 Quick Start

Najkraći vodič za start i osnovni rad.

  1. Pokreni server: python3 server/paprikas_server.py --host 127.0.0.1 --port 8015
  2. Otvori: http://127.0.0.1:8015/
  3. Na Početnoj izaberi željeni modul.
  4. Za nove recepte koristi 🧰 Content Studio.
  5. Za pregled baze koristi 📖 Recipe Atlas.
Savet: radi iz Početne — ona je komandni centar aplikacije.
