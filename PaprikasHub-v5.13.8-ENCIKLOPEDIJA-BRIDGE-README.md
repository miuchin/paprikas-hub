# Paprikas Hub v5.13.8 - Enciklopedija bridge build

Ovaj build dodaje read-only Biblioteka / Enciklopedija modul u Paprikas Hub.

## Glavne novine
- novi home launcher: Biblioteka
- novi TAB: Biblioteka / Enciklopedija
- ugrađena HTML knjiga za offline/local pregled
- bridge JSON sa jasnom podelom:
  - Paprikas Hub relevantno
  - SINET relevantno
- seed za budući atlas biljaka i namirnica
- SINET export bridge za review/import

## Pokretanje
```bash
cd paprikas-hub
bash start.sh
```
