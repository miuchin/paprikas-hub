## GitHub Release text
```md
# Enciklopedija narodnih metoda lečenja - Final static release

Ovo izdanje donosi deploy-ready statički paket pripremljen za **GitHub Pages** i **Netlify**.

## Sadržaj paketa
- `index.html` - glavni prikaz knjige
- `pregled.html` - pregled poglavlja i brza navigacija
- `data/structured.json` - strukturirani izlaz knjige
- `data/sinet_ready_katalog.json` - kurirani SINET katalog
- `data/oblasti/` - podpaketi po oblastima
- `data/tipovi/` - podpaketi po tipovima
- `data/review_candidates.json` - minimalni preostali review set
- `source/enciklopedija_narodnih_metoda_lecenja_original.pdf` - originalni PDF izvor
- `README.md`, `README_SR.md`, `README_EN.md`
- `MANIFEST.md`, `QC_REPORT.md`, `RELEASE_NOTES.md`
- `NETLIFY_DEPLOY_CHECKLIST.md`
- `NOVI_CHAT_REKAPITULACIJA.md`

## Ključne osobine
- responsive HTML prikaz
- navigacija po poglavljima i stranama
- pretraga i filtriranje
- strukturirani JSON za dalji rad
- SINET-ready katalog za uvoz i obradu
- spremno za static hosting bez build koraka

## Trenutni katalog
- recepti / protokoli: 50
- biljke i namirnice: 59
- tegobe: 29
- opšti saveti: 9

## Napomena
Paket je regenerisan iz stabilnih artefakata sačuvanih u ovoj sesiji kako bi ponovo bio dostupan za preuzimanje i objavu.
```
